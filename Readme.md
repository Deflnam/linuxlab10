# Matrix Solver

## Название программы

**Matrix Solver** – программа для решения систем линейных алгебраических уравнений (СЛАУ) методом Гаусса.

## Что делает программа

Программа решает систему уравнений вида **A × x = b**, где:
- **A** – квадратная матрица коэффициентов
- **b** – вектор правой части
- **x** – вектор неизвестных (решение)

Программа имеет три версии вычислительного ядра на C++:
1. **Наивная** – без оптимизаций
2. **Блочная векторизация** – с использованием AVX инструкций
3. **Блочная векторизация + выравнивание** – с оптимизацией памяти

Головная программа написана на Python и предоставляет графический интерфейс для:
- Генерации тестовой матрицы
- Выбора версии решателя
- Отображения времени решения и результата

## Как запустить

### 1. Установите зависимости

```bash
sudo apt update
sudo apt install -y g++ python3 python3-tk python3-numpy make
```

### 2. Соберите библиотеки

```bash
make
```

### 3. Запустите программу

```bash
python3 main.py
```

### Альтернативный запуск (без графики)

```bash
python3 benchmark.py
```
